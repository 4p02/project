# Web extension for summarily

# PLEASE NOTE THAT THE EXTENSION IS NOT YET FUNCTIONAL AND JUST BOILERPLATE CODE WITH MISSING FILES

## Installation

Install web-ext using npm package manager
```
npm install --global web-ext
```

## Usage
```
web-ext run
```
